# Gank

>此项目旨在学习 React Native，暂时只完成iOS平台部分

[干货集中营](http://gank.io)非官方App

- 系统及开发工具：MacOS + WebStorm + Xcode
- react-native版本: **0.40**

## 如何运行

1. git clone https://github.com/OceanOver/Gank.git

2. cd Gank

3. npm install

4. react-native link

5. react-native run-ios
